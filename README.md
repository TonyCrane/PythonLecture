## Build this slide

1. Install `nodeppt` via `npm`

    ```sh
    $ npm install -g nodeppt
    ```
2. Start local server to preview

    ```sh
    $ nodeppt serve index.md
    ```
3. Build slide

    ```sh
    $ nodeppt build index.md
    ```
